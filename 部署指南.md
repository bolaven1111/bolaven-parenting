# 布拉万的育儿大师 - 部署指南

## 需要什么
- 一个 **GitHub 账号**（免费：github.com）
- 一个 **Vercel 账号**（免费：vercel.com，用 GitHub 登录）
- **DeepSeek API Key**（你已经有）

---

## 部署步骤（全程约 10 分钟）

### 第一步：把代码上传到 GitHub

1. 打开 https://github.com ，登录你的账号
2. 点右上角 **+** → **New repository**
3. Repository name 填：`bravan-parenting`
4. 选 **Public**（公开，免费计划不需要付费）
5. 其他默认，点 **Create repository**
6. 页面会出现上传文件的入口，把项目文件夹里的文件全部拖进去上传：
   - `index.html`
   - `vercel.json`
   - `package.json`
   - `.gitignore`
   - `api/search.mjs`
   - `api/stats.mjs`
   - `knowledge/index.json`
   - `knowledge/美国儿科学会/` 里面的所有 .md 文件（共36个）

### 第二步：在 Vercel 部署

1. 打开 https://vercel.com ，用你的 GitHub 账号登录
2. 点 **Add New...** → **Project**
3. 在列表里找到刚才创建的 `bravan-parenting`，点 **Import**
4. 在 **Environment Variables** 区域，添加：
   - **Name**: `DEEPSEEK_API_KEY`
   - **Value**: `sk-740f3899611f48a39d747565362dba4f`
5. 其他全部默认，点 **Deploy**
6. 等一两分钟，出现 **Congratulations!** 就部署完成了
7. 点 **Continue to Dashboard**，看到那个绿色的域名就是你的网站地址

### 第三步：发给家人

把 Vercel 给你的域名（比如 `bravan-parenting.vercel.app`）直接发给家人，打开就能用。
